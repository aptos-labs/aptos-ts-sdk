# Distribution

Recommended: store this skill in repository path:

`skills/aptos-ts-sdk-transaction-workflows`

## Standard Install Route (GitHub Path)

Use the skill-installer script:

`scripts/install-skill-from-github.py --repo <owner>/<repo> --path skills/aptos-ts-sdk-transaction-workflows`

Behavior:
- Installs to `$CODEX_HOME/skills/aptos-ts-sdk-transaction-workflows` (default `~/.codex/skills/...`).

## Manual Fallback

Copy folder directly:

`skills/aptos-ts-sdk-transaction-workflows -> ~/.codex/skills/aptos-ts-sdk-transaction-workflows`

## Post-Install

Restart Codex so the new skill is picked up.
